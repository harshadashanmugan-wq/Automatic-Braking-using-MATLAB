# Automatic-Braking-using-MATLAB

**Automatic-Braking-using-MATLAB** is an ADAS-based automatic braking system that uses **Time-to-Collision (TTC)** calculations to provide **real-time simulation** and **visual alerts**. The system detects potential collisions and triggers braking to ensure vehicle safety.

## Features
- Real-time detection of obstacles using TTC.
- Visual alerts for different danger levels.
- Automatic braking system simulation in MATLAB.
- Clear representation of safe, warning, and danger zones.

## Folder Structure
- `Architecture/` – Block diagrams and system architecture files.
- `Brake applied/` – Simulation results showing braking applied.
- `Danger Zone/` – Visuals of danger zones.
- `Safe zone/` – Visuals of safe zones.
- `Demo/` – Demonstration files.
- `Feature Logic.docx` – Detailed feature logic.
- `Warning graph.jpeg` – Graphs indicating warning levels.
- `brake.pdf` – Brake application details.

## How to Run
1. Open MATLAB.
2. Load the project files.
3. Run the main simulation script.
4. Observe visual alerts and automatic braking response.
